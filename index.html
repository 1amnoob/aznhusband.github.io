Repository addon

Instructions for adding this repo

  • Go to the Kodi file manager
  • Add a source. Source is https://aznhusband.github.io/repo Call it whatever name you'd like.
  • In Addons, install an addon from zip. When it asks for the location, select whatever you called it in the previous step, and install repository.aznhusband-kodi-repo-1.0.1.zip
  • Then install addon from repo, and select the "aznhusband" repo.

For now, this plugin depends on UrlResolver. You will need to install that from elsewhere.
